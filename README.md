# My Portfolio (个人作品集展示网站)

## 目录结构

```
/portfolio-site
├── public/
│   └── favicon.ico
├── pages/
│   └── index.tsx      // 首页逻辑
├── styles/
│   └── globals.css
├── data/
│   └── projects.json  // 项目数据文件
├── components/
│   └── ProjectCard.tsx
├── tailwind.config.js
├── next.config.js
├── vercel.json (可选)
└── package.json
```

## 启动方式

```bash
npm install
npm run dev
```
